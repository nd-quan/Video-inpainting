Version 2 Formula

For each clip C_k, the shared background seed is computed as the mean of the initial random noise of all frames in the clip: 

z_shared_bg,k = mean({z_T^i | i in C_k})

Each frame then uses a constrained BG residual: 

z_hat_T^i = z_T^i * ROI_mask^i + (z_shared_bg,k + r_i^BG) * BG_mask^i
||r_i^BG||_2 <= sigma

Pipeline Changes

New pipeline arguments:

use_shared_bg_noise: bool = False
shared_bg_noise: Optional[torch.FloatTensor] = None
shared_bg_noise_strength: float = 1.0
use_pgd_bg_residual: bool = False
residual_sigma: float = 1.0
residual_init_scale: float = 1.0

Key implementation behavior:

  • original_mask is converted inside the pipeline so that BG = 1 and ROI = 0.
  • If shared_bg_noise=None, the pipeline computes z_shared_bg from the clip batch mean.
  • residual_bg = noise - z_shared_bg is applied only on BG and projected to an L2 ball.
  • The scheduler stores residual_bg, bg_mask_latent, pgd_lr, pgd_steps, and the PGD denoising window.
  • During DDIM sampling, CGE stores the raw gradient and the scheduler updates residual_bg with PGD.
  • ROI keeps the original per-frame noise.

PGD Residual Update

The current implementation performs residual updates inside CustomDDIMScheduler.step(). The CGE function returns the normal guidance update and also stores the raw gradient before guidance_scale is applied. 

raw_grad = dL / dz
if pgd_start_step <= denoise_step < pgd_end_step:
residual_bg = residual_bg - pgd_lr * raw_grad * BG_mask
residual_bg = project_l2_ball(residual_bg, residual_sigma) * BG_mask
prev_sample = prev_sample + (residual_bg_new - residual_bg_old) * BG_mask

This is a practical PGD residual update coupled to the DDIM/CGE loop. It avoids an expensive outer loop that would rerun the entire diffusion process for every PGD iteration.

The current test default uses PGD_START_STEP=10 and PGD_END_STEP=20, so CGE/PGD is delayed until the decoded prediction is less noisy. The end index is exclusive.

Test Script Changes

The test script now imports the PGD pipeline and CGE scheduler:

from diffusers.pipelines.brushnet.pipeline_brushnet_sharedNoise_PGD_v0 import StableDiffusionBrushNetPipeline
from diffusers.schedulers.scheduling_ddim_CGE import CustomDDIMScheduler, cond_fn

The scheduler is changed from normal DDIM to CGE DDIM:

pipe.scheduler = CustomDDIMScheduler.from_config(pipe.scheduler.config)

For each clip, the script prepares a batch of frames and activates CGE:

pipe.scheduler.x_lr = torch.cat(x_lr_tensors, dim=0)
pipe.scheduler.mask = torch.cat(mask_tensors, dim=0)
pipe.scheduler.decoder = pipe.vae.decode
pipe.scheduler.cond_fn = cond_fn

result = ip_model.generate_fgbg(

VVC Runtime Fix

The original VVC binary at model_naeun/paper/VVC_Python/vvencFFapp fails on this machine because it requires newer GLIBC and GLIBCXX versions. This caused vvc_func_512 to miss output_*.png and crash with FileNotFoundError.

The CGE scheduler now defaults to the compatible encoder:

/media/ssd1/ndquan/TGDR/VVC/vvencFFapp

You can override it when running with:

CGE_VVC_ENCODER=/path/to/vvencFFapp

The VVC temporary directory is now kept only when VVC fails, so the remaining files can be inspected. Set CGE_KEEP_VVC_LOGS=1 to keep successful VVC folders as well.

Important Note

This implementation now updates residual_bg with PGD during DDIM steps using the raw CGE gradient. It is not an outer-loop optimizer that reruns the full denoising process several times; instead, it couples PGD residual updates to the existing sampling loop for lower memory cost.
